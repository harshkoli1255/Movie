## .env Setup Instructions

To set up your environment variables, please follow the steps below:

1. **Clone the repository:**  
   Run the following command to clone the repository:  
   ```bash
   git clone https://github.com/harshkoli1255/Movie.git
   ```

2. **Navigate to the directory:**  
   Change into the project directory:  
   ```bash
   cd Movie
   ```

3. **Create a .env file:**  
   In the root of the project, create a new file named `.env` and add the following variable:
   ```plaintext
   MONGODB_URL=your_mongodb_connection_string_here
   ```

4. **Install dependencies:**  
   Use npm to install all necessary dependencies:  
   ```bash
   npm install
   ```

5. **Run the application:**  
   Finally, start the application with:  
   ```bash
   npm start
   ```

Make sure to replace `your_mongodb_connection_string_here` with your actual MongoDB connection string.
